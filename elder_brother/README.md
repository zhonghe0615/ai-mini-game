# 投篮小游戏

这是一个简单的 HTML5 Canvas 投篮游戏。

## 如何运行

直接用浏览器打开 `index.html` 即可。

## 替换卡通人物

您可以自己画一个，或者从网上找一张您喜欢的图片来替换！

### 简单模式（只需一张图）

1.  准备一张透明背景的图片，命名为 `player.png`。
2.  放入 `elder_brother/` 文件夹。
3.  人物会一直保持这个样子。

### 进阶模式（带投篮动作）

如果您想让人物动起来，可以准备两张图片：

1.  **准备姿势**：命名为 `player_idle.png`（持球准备）。
2.  **投篮姿势**：命名为 `player_shoot.png`（手伸直投篮）。
3.  将这两张图放入 `elder_brother/` 文件夹。

游戏会自动检测：如果找到了这两张图，就会优先使用它们，实现投篮时的动作切换！

## 替换篮球

1.  准备一张篮球图片，命名为 `ball.png`。
2.  放入 `elder_brother/` 文件夹。

## 替换音效

将您找到的 MP3 音效文件放入 `elder_brother/` 文件夹，并按照以下名称命名：

1.  `score.mp3` - 进球后的庆祝声音（比如“刷网声”或“欢呼声”）
2.  `rim.mp3` - 撞击篮筐（金属）的声音
3.  `board.mp3` - 撞击篮板（木头）的声音
4.  `floor.mp3` - 撞击地板或墙壁的声音

**注意：** 由于浏览器的安全策略，可能需要您先与页面交互（点击一下）才能播放声音。
